  2. 生成 CA 证书请求

  3. 生成 CA 根证书

    jabin@dev-ubuntu [02:02:46] [~/ssl]
    -> % openssl x509 -req -in local.csr -extensions v3_ca -signkey local.key -out local.crt
    Signature ok
    subject=C = CN, ST = GD, L = SZ, O = Internet Widgits Pty Ltd, OU = dev, CN = ubuntu, emailAddress = dev@dev.com
    Getting Private key
